# mip-mfk-source

mip-mfk-source 统计页面来源

标题|内容
----|----
类型|通用
支持布局|responsive,fixed-height,fill,container,fixed
所需脚本|https://c.mipcdn.com/static/v1/mip-mfk-source/mip-mfk-source.js

## 示例

### 基本用法
```html
<mip-mfk-source src="https://m.mfk.com/"></mip-mfk-source>
```

## 属性

### src
说明：数据存储地址
必选项：是
类型：String
